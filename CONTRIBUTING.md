# Contributing

Contributions are welcome and will be fully credited.

## Pull Requests

- **[PSR-12 Coding Standard](https://www.php-fig.org/psr/psr-12/)** - Use Laravel Pint to fix code style.
- **Add tests** - Your patch won't be accepted if it doesn't have tests.
- **Document any change in behavior** - Make sure the `README.md` is updated.
- **One pull request per feature** - If you want to do more than one thing, send multiple pull requests.

## Running Tests

```bash
composer test
```

## Running Code Style Fixer

```bash
composer format
```

## Running Static Analysis

```bash
composer analyse
```

**Happy coding!**
